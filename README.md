# Promoter-Score
O NPS® ou Net Promoter Score é uma métrica desenvolvida para medir os níveis de lealdade do cliente, criada em 2003 pelo consultor da Bain &amp; Company, Fred Reichheld, em colaboração com a Satmetrix.
